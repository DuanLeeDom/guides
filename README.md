# Guides

Este repositório tem como objetivo reunir **guias de ajuda** para consulta rápida e aprendizado.  
Aqui serão armazenados arquivos como:

- Documentações em `.md`
- Livros e materiais em `.pdf`
- Outros conteúdos úteis para estudar ou revisar assuntos específicos

O foco é facilitar o acesso a materiais de apoio técnicos e educacionais em um só lugar.
